Accessing Jupyter Notebook running on a Google Compute Instance

08-22-2018
TL;DR

To acces Jupyter Notebook running in GCloud Instance

  • Make the compute instance static from External IP Addresses in the VPC Network menu
  • Add a Firewall rule for all instances
  • 0.0.0.0/0 for Source IP ranges and tcp: for Specified protocols and ports where is the number you used above.
  • Hit create Running jupyter-notebook –port=
