Mobius - Visual Programming Web App
Product Lead
Built with Angular, Mobius offers a visual programming environment in the browser, merging flow-based and textual programming paradigms. It is an easy-to-use, open source platform, customizable and modular data viewers and function libraries.
Mobius was also used for Extruded Cities - a 5-day workshop, conducted during the CAADRIA Conference - 2018, in Beijing, China. Students learned to process and visualize 3D Geospatial Data using the Mobius Visual Programming Environment. They then used Mobius' publishing capabilities to publish they work as interactive websites on Github.
